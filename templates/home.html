All about my project

UI Component

This project uses Bootstrap5 for most of its styling, but also includes css in its static files
I used django crispy forms for the forms on this webiste as it makes the forms easier to read and more user friendly

Web Application

A user can log in, log out, reset their password (if they know their email!) and change their password if they're logged in
There are multiple database models used in this project:

  • CutsomUser
  • Profile
  • Movie
  • Comment
A user who is staff is able to Update, Create, Delete movies from the site
A user is also able to view all the movies and comments
Database population was made easy by the use of json
A one-to-many relationship was used in the Movie app
A movie can have many comments
A one-to-one relationship was used in the Accounts app
A user can have one and only one Profile
I used static files for a user profile and for The Image on the Homepage
A user is created based on a Custom User Model, and can have one and only one profile
User authentication is used not only in the templates "if user.is_staff" but is also used in Views.py using mixins
A staff user can add a movie to the database, and can even upload a cover for the movie

Version Control and Git

This project is uploaded to Github and uses git to track changes to all of the files
I have two branches on this project:

  • main
  • old-state

Features I'm really proud of

  • When search results are returned, I figured out how to display the number of results I had
  • A staff user is able to upload an image for a new movie without having to go to admin
  • My movie list page i think is designed really well and makes good use of the media
  • On the movie list page, the number of comments is counted and shown to the user

Features I hope to improve in the future

  • At the moment a user cannot create their profile on the webpage it has to be done manually in the backend by a staff member
  • A comment can only be added to a movie in the admin page
  • I also think with more time I would've liked to add a categories model so that I could sort the movies out into their genres
